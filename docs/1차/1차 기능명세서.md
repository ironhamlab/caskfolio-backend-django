# Caskfolio 기능 명세서

#### 2026.3.3. 작성, 수정할 수도 있음

## 서비스 개요
| 항목 | 내용 | 
| --- | --- |
| 서비스명 | Caskfolio | 
| AI 큐레이터 |  Whiscurator | 
| 플랫폼 | 반응형 웹 (PWA 전환 예정), 모바일 우선 |
| 테마 | 다크(기본) / 라이트 전환 지원 | 
| 주요 색상 | Amber Night (#C97B2F) / Oak & Cream (#8B4513) |


<br/>

## 섹션 구조 (하단 탭바 4개)

🥃 위스키 아카이브 : 탐색, 검색, 필터, 도슨트

🤖 위스큐레이션 : AI 큐레이터 Whiscurator 상담

📝 테이스팅 노트 : 노트 작성 및 둘러보기

👤 마이페이지내 바, 내 기록, 설정

### A. 위스키 (아카이브)
#### A-1. 위스키 목록
| 항목 | 내용 |
| --- | --- |
| 기능 | 전체 위스키 목록 조회 |
| 검색 | 이름, 증류소명 연관 검색 |
| 필터 | 스타일 / 지역 / 캐스크 타입 / 피트 강도(None·Light·Medium·Heavy) / 숙성 연수 / 맛 계열(Smoky·Sweet·Fruity·Spicy·Woody·Floral) — 각 분류별 복수 선택 가능 |
| 정렬 | 이름순 / 최신 등록순 / 테이스팅 노트 많은 순 |

#### A-2. 위스키 상세 (도슨트)
| 항목 | 내용 |
| --- | --- |
| 기본 정보 | 이름, 증류소, 스타일, 지역, ABV, 숙성 연수, 캐스크 타입 |
| Radar Chart | Smoky / Sweet / Fruity / Woody / Spicy / Floral 6축 시각화 |
| 서빙 가이드 | Neat / On the Rocks / Splash / Soda 방식별 설명 |
| 도슨트 콘텐츠 | 역사, 바텐더 팁, 안주 페어링 | 
| 캐스크 설명 | 해당 캐스크 타입이 향미에 미치는 영향 |
| 액션 | 장식장 추가 / 위시리스트 추가 / 테이스팅 노트 작성 바로가기 |

<br/>

### B. 위스큐레이션 (Whiscurator)
#### B-1. AI 큐레이터 상담
| 항목 | 내용 |
| --- | --- |
| 기능 | 위스키 DB 기반 RAG 챗봇 |
| 상담 유형 | 상황 기반 추천 ("오늘 스트레스 받았는데 뭐 마실까") / 위스키 궁금증 해소 ("셰리 캐스크가 왜 달콤해?") |
| UI | 일반 챗봇 형식, 이전 대화 세션 목록 유지 |
| 추천 방식 | 추천 근거 함께 제시, 추천 위스키 카드 인라인 노출 |
| 데이터 소스 | Whisky 모델의 description, tasting_notes, history, bartender_tip 등 |

<br/>

### C. 테이스팅 노트
#### C-1. 노트 작성
| 항목 | 내용 |
| --- | --- |
| 진입 | 플로팅 버튼으로 언제든 접근 가능 |
| 위스키 선택 | DB에서 검색 후 선택 |
| 태그 | 자유 입력 (인스타그램 태그 방식), 태그당 20자 이내 / 최대 10개 |
| 노트 | 줄글 자유 작성 (선택) |
| 별점 | 0.5 단위, 5.0 만점 |
| 시음 날짜 | 기본값 오늘, 직접 수정 가능 (과거 기록 가능) |
| 공개 여부 | 공개 / 비공개 선택 (기본값은 앱 설정에서 지정) | 
| 중복 작성 | 같은 위스키 여러 번 기록 가능 (서빙 방식, 날짜 다를 수 있음) |

#### C-2. 둘러보기
| 항목 | 내용 |
| --- | --- |
| 기본 탭 | 내 기록 (피드 형식) | 
| 둘러보기 탭 | 공개 처리된 다른 유저 노트 |
| 필터 | 위스키 스타일 / 지역 기반 필터링 |
| 검색 | 위스키명 연관 검색 |
| 유저 클릭 | 해당 유저가 작성한 공개 노트 목록 확인 가능 |
| 수정/삭제 | 본인 노트만 수정 및 삭제 가능 |

<br/>

### D. 마이페이지
#### D-0. 프로필
| 항목 | 내용 |
| --- | --- |
| 표시 정보 | 닉네임, 한 줄 소개 | 
| 요약 통계 | 마셔본 위스키 N개 / 위시리스트 N개 / 작성 노트 N개 |
| 설정 버튼 | 우측 상단 ⚙️ 버튼으로 진입 |

#### D-1. 내 바
| 항목 | 내용 |
| --- | --- |
| 장식장 | 마셔본 위스키 컬렉션. Collection에 추가 시 위시리스트에서 자동 삭제 | 
| 위시리스트 | 마셔보고 싶은 위스키 저장. 미리보기 후 전체보기 이동 |

#### D-2. 내 기록
| 항목 | 내용 |
| --- | --- |
| 활동 로그 | 내가 작성한 테이스팅 노트 리스트 형식, 수정/삭제 가능 |
| 나의 여정 | 시간 흐름에 따른 취향 변화 시각화 (탐색한 지역, 피트 강도 변화 등) |

#### D-3. 설정
| 항목 | 내용 |
| --- | --- |
| 개인정보 수정 | 닉네임, 한 줄 소개, 이메일, 비밀번호 변경, 회원 탈퇴 | 
| 앱 설정 | 테마 (다크/라이트), 새 노트 기본 공개 여부 |

<br/>

### E. 인증 (Auth)
| 항목 | 내용 |
| --- | --- |
| 가입 | 이메일 + 비밀번호 + 닉네임 |
| 로그인 | JWT 인증 |
| 탈퇴 | 소프트 딜리트 (deleted_at 기록, 데이터 즉시 삭제 안 함) |

## 개발 우선순위 (MVP)
| Phase | 범위 |
| --- | --- |
| Phase 1 | User/Auth + Whisky 모델 + 어드민 데이터 입력 |
| Phase 2 | 아카이브 목록/검색/필터 + 상세 페이지 |
| Phase 3 | 테이스팅 노트 CRUD + Collection/Wishlist |
| Phase 4 | 마이페이지 (내 바, 활동 로그, 여정 시각화) | 
| Phase 5 | Whiscurator (RAG + 대화 세션) |
