# my-whisky-project

2026년의 버킷리스트 중 하나가 '매달 새로운 술 마셔보기' 입니다.

여러 주종 중 가장 마음에 들었던 **위스키**를 주제로,

위스키 추천 및 기록용 웹 사이트를 간단하게 만들어보고자 합니다.


<br/>


## 🔥 Project 소개

* 서비스 명 : 위스큐레이터(Whiscurator)
* 위스키에 대해 큐레이팅 받는 느낌으로 서비스를 사용할 수 있으면 좋을 것 같아서!



<br/>
<br/>
<br/>

## 🔥 branch 전략

* main(master): 배포 가능한 상태 관리 브랜치
* develop(개발): feature 브랜치 병합용 브랜치
* feature(기능): 기능 개발 및 버그 수정용 브랜치



<br/>

## 🔥 git commit 컨벤션

| 작업 타입 |	작업내용 |
| --- | --- | 
| feat	| 해당 파일에 새로운 기능이 생김 |
| fix	| 버그 수정 | 
| docs	| 문서 수정 | 
| refactor	| 코드 리팩토링 | 
| style	| 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우 |
| chore	| 빌드 업무 수정, 패키지 매니저 수정 |
